# Referências de Design - EduTrack AI

## Template 1: Education Dashboard
- **Link:** https://www.figma.com/design/WRvXcR6UOy9SpZvivhUhcU/Education-Dashboard--Community-
- **O que tem de bom:**
  - Cards com números de Students, Departments, Courses e Modules
  - Tabela de "Star Students" com nome, nota e porcentagem
  - Gráficos de "Student Activity"
  - Navegação lateral com Dashboard e Students

- **Por que é útil para o EduTrack:**
  - Dashboard com cards de estatísticas para visão geral dos alunos
  - Tabela de destaques pode virar ranking de desempenho

## Template 2: Educational Dashboard
- **Link:** https://www.figma.com/design/OUk2d6NZulnNZkrGkaj1Rk/Educational-Dashboard--Community-
- **O que tem de bom:**
  - Mensagem de boas-vindas personalizada ("Welcome back, Ayo")
  - Seção "My Progress" com dias da semana
  - "Completion Progress" com barras de progresso
  - "Upcoming Activities" para próximas tarefas
  - Versões Light mode e Dark mode

- **Por que é útil para o EduTrack:**
  - Progresso semanal ideal para acompanhamento do aluno
  - Atividades futuras podem virar lista de tarefas/prazos

---

## Maior sacada que tive com os templates:

Um bom dashboard educacional precisa de:
1. **Visão geral rápida** (cards com números)
2. **Destaques/ranking** para engajar alunos
3. **Acompanhamento de progresso** (barras ou gráficos)
4. **Próximas atividades** para organizar o aluno
5. **Mensagem pessoal** para criar conexão

Percebi que posso combinar o melhor de cada template: os cards estatísticos do Template 1 com o acompanhamento semanal do Template 2.
